/creditbundles

/creditbundles get post

get
post

get /creditbundles

  • Request
  • Response

Query Parameters

  • user: (string)

    Assigned user

    Example:

    john@gmail.com
  • include_expired: (boolean - default: false)

    Include expired bundles

    Example:

    true
  • include_empty: (boolean - default: false)

    Include empty (quantity = 0) bundles

    Example:

    true

HTTP status code 200

post /creditbundles

  • Request

Body

Media type: application/json

Type: object

Properties
  • expire_date: required (date-only)

    Expiration date

    Example:

    2019-01-22
  • quantity: required (integer)

    Credit Quantity

    Example:

    100
  • assigned_to: required (array of items)

    Assigned Users

Example:

{
  "expire_date": "2019-01-22",
  "quantity": 100,
  "assigned_to": [
    "mike@gmail.com",
    "sarah@gmail.com"
  ]
}
